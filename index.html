AI Builder Evaluation Workbench

AI Builder 평가 페이지

Resume PDF를 업로드하면 KPMG AI Builder 역할 기준에 맞춰 6개 역량 축의 근거를 추출하고, 정량 점수와 평가자 calibration 화면을 제공합니다.

Resume-grounded RAG레쥬메 문장을 축별로 검색해 근거를 먼저 분리합니다.
6-axis scoring0~10점으로 점수화하고 6각형 차트로 비교합니다.
Human calibration애매한 부분은 평가자가 점수와 코멘트를 조정합니다.

6개 평가 축

학교나 회사 브랜드보다 Resume 안의 실제 근거를 기준으로, 각 축을 0~10점으로 평가합니다.

Problem Framing

모호한 문제를 정의하고 MVP 범위, 성공 기준, tradeoff를 세운 근거를 봅니다.

Practical Building

프로토타입, 자동화, API, dashboard 등 실제 작동 산출물을 만든 경험을 봅니다.

AI Leverage

LLM, RAG, agent, prompt evaluation을 문제 해결 workflow에 적용한 방식을 봅니다.

Enterprise Judgment

보안, privacy, governance, human approval, 운영 리스크를 고려했는지 봅니다.

Stakeholder Thinking

비즈니스, IT, legal, 사용자 등 여러 이해관계자 요구를 조율한 근거를 봅니다.

Communication & Growth

복잡한 내용을 설명하고 빠르게 배우며 문서화, 데모, onboarding까지 연결한 경험을 봅니다.

평가 파이프라인

AI는 Resume 근거 추출과 정리만 돕고, 최종 판단과 calibration은 사람이 수행합니다.

1
PDF Intake

Resume PDF에서 텍스트를 추출하고 분석 가능한 문장 단위로 나눕니다.

2
Evidence Retrieval

6개 axis별로 관련 문장을 검색해 RAG evidence 후보를 만듭니다.

3
Scoring

근거의 구체성, ownership, impact, enterprise relevance를 기준으로 점수화합니다.

4
Comparison Set

같은 axis의 다른 지원자 평가 사례와 비교해 평가 일관성을 확인합니다.

5
Human Calibration

평가자가 점수와 메모를 조정하고 후속 인터뷰 질문을 정합니다.

데모 목적의 프론트엔드 프로토타입입니다. 실제 운영에서는 API 키와 Resume 원문을 브라우저에서 직접 OpenAI로 보내지 말고 Netlify Function 같은 서버 프록시와 보안 검토를 사용하세요.
PDF를 올리거나 샘플 분석을 실행하세요.
